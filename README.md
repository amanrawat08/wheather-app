 # 🌦️ Weather App

A simple and responsive **Weather Application** that allows users to check real-time weather conditions of any city using a weather API.

---
## 📌 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, wind speed
* ☁️ Shows weather conditions (Cloudy, Sunny, Rainy, etc.)
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Fast and clean UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Weather API (e.g., OpenWeatherMap)

## ⚙️ How It Works

1. User enters a **city name**
2. App sends request to **Weather API**
3. API returns weather data (JSON)
4. Data is displayed dynamically on UI

Weather apps typically use APIs to fetch **real-time data like temperature, humidity, and wind speed** ([GitHub][1])

---

## 🔑 API Setup

1. Get your API key from:
   👉 [https://openweathermap.org/api](https://openweathermap.org/api)

2. Replace API key in your JavaScript file:

```js
const apiKey = "YOUR_API_KEY";
```
